Schedule Formatting

  ┌───────── min (0 - 59)
  │ ┌────────── hour (0 - 23)
  │ │ ┌─────────── day of month (1 - 31)
  │ │ │ ┌──────────── month (1 - 12)
  │ │ │ │ ┌───────────── day of week (0 - 6)
  │ │ │ │ │
  │ │ │ │ │
  * * * * *

Where * indicates every

Tasks run as the apache user, not intended for root tasks.
